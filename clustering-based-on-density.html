3. DBSCAN

   Definations. - Eps: Threshold of distance for measuring density. - Minpt: Minimum number of points to define a core point. - Core points: Points with the count of points within distance of Eps greater than Minpt. - Border points: Points fall into the neighbor of core points. - Noise points: Points left.

   Time and space complexity - Time: O(m^2). Might be reduced to O(m*logM) if kd-trees are used for find nearest k neighbors. - Space: O(m). Only need to store whether it is a core point or border point or noise.
